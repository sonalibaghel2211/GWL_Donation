# Donations: Recurring & Receipts

Donations: Recurring & Receipts is a Shopify app that enables merchants to seamlessly collect donations across their store. It allows customers to contribute to campaigns or causes directly from product pages, cart, or checkout, ensuring a smooth and integrated donation experience.

---

## Features

### Donation Management

* Automatically creates and syncs a Shopify donation product
* Supports predefined donation amounts (e.g., $10, $25, $50)
* Allows custom donation input
* Enables recurring donations (weekly, monthly, yearly)
* Supports multiple donation campaigns

---

### Receipts and Compliance

* Sends automatic donation receipts after checkout
* Allows re-sending receipts from the admin panel
* Supports customizable email templates
* Automatically tags customers as Donor or Recurring Donor
* Tags orders as donation-related

---

### Storefront Integration

* Adds donation widgets to product, cart, checkout, and home pages
* Supports “In Honor / Memory Of” donation fields
* Includes round-up donation functionality in cart
* Displays campaign goal progress
* Fully customizable using Online Store 2.0 App Blocks

---

### Analytics and Dashboard

* Tracks total donations and revenue
* Provides campaign-level performance insights
* Displays average donation values
* Offers detailed reporting

---

### Backend and Operations

* Supports tier-based pricing plans (Free, Pro, Enterprise)
* Integrates Shopify webhooks for real-time updates
* Handles subscriptions and order events
* Designed for scalability and performance

---

## How It Works

### Customer Flow

1. Customer visits the store
2. Donation widget is displayed
3. Customer selects or enters a donation amount
4. Donation is added to the cart
5. Customer completes checkout
6. Receipt is generated and sent automatically

### Merchant Flow

1. Install the app
2. Create donation campaigns
3. Configure donation settings
4. Add the widget using theme editor
5. Monitor performance through the dashboard

---

* Shopify Embedded App
